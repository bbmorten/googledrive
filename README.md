# Google Slides API

## Prepare the environment

```text
mkdir spread-to-slides
cd spread-to-slides
virtualenv venv
source venv/bin/activate
```

## Installing Google Client Libraries

```code
  pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
```
